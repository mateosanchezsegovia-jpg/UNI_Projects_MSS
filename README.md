## 📌 Important Notes 

Before exploring the code, please keep the following in mind:

* **Language:** While this README and the general structure of the repository are in English, some projects, code comments, and documentation are written in Spanish. This is due to the specific grading rubrics and requirements of the courses at the time they were developed.
* **Legacy Dependencies:** A few of the projects—especially those from my early years in the degree—were built in highly specific local environments. Consequently, some external dependencies or configuration files might not be included in the repository. These projects are archived here as a reflection of my learning curve and progression rather than as plug-and-play software.
